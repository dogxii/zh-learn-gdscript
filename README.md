# Learn GDScript From Zero

Learn GDScript From Zero 是一款免费的开源应用程序，供绝对初学者学习使用 Godot 的 GDScript 语言进行编程。

这是一门适合初学者的编程课程。

这个系列与Godot 4兼容。

这个系列翻译自于 [原项目](https://github.com/GDQuest/learn-gdscript)。

## 快速开始
